# The Running Dead
 2D Top Down zombie shooter game made using C/C++ SFML library. 

Made by Gregory Jabido and Gunnar Oye

Final project for CPTS 122 Spring 2024

W - move up

A - move left

S - move down

D - move right

Mouse to aim

Left click to shoot
